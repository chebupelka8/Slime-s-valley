# How to play

Moving:
left: KEY "left" or KEY "a"
right: KEY "right" or KEY "d"
jump: KEY "SPACE"

Shoot: Left mouse button
Show cursor: Right mouse button

Beware of spikes and enemies!!!

If you dead, you return to the starting position
